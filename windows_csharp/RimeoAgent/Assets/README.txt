Place the following files in this folder before building:

  rimeo.ico   — application icon (convert rimeo1024.png → .ico using any online converter)
                 The .ico file must contain at least 16x16, 32x32, 48x48, 256x256 sizes.

  cloudflared.exe  (optional) — bundled Cloudflare tunnel binary
                 Download from: https://github.com/cloudflare/cloudflared/releases/latest
                 If not present, the agent will search PATH and common install locations.
